Политика конфиденциальности

Последнее обновление: 2 марта 2026 г.

1. Введение

Настоящая Политика конфиденциальности описывает, как приложение Hisob ("мы", "наше приложение") собирает, использует и защищает вашу личную информацию.

Используя наше приложение, вы соглашаетесь с условиями настоящей Политики конфиденциальности.

2. Сбор информации

Наше приложение собирает следующие типы информации:

  • Учетная информация: адрес электронной почты, имя пользователя при регистрации через Firebase Authentication
  • Финансовые данные: информация о доходах, расходах, долгах (хранится локально на вашем устройстве)
  • Данные о товарах: информация о товарах, остатках, продажах (хранится локально)
  • Технические данные: информация об устройстве, версия приложения для обеспечения работы приложения

3. Использование информации

Мы используем собранную информацию для:

  • Предоставления и улучшения функциональности приложения
  • Обеспечения работы учетной записи пользователя
  • Синхронизации данных между устройствами (при наличии подписки)
  • Отправки уведомлений о важных обновлениях
  • Обеспечения технической поддержки

4. Хранение данных

Локальное хранение: Все ваши финансовые данные, информация о товарах, долгах и задачах хранятся локально на вашем устройстве в зашифрованном виде.

Облачное хранение: При использовании функции резервного копирования через Telegram, зашифрованные копии ваших данных сохраняются в вашем личном Telegram-боте. Мы не имеем доступа к этим данным.

Firebase: Информация о подписках и учетных записях хранится в защищенной базе данных Firebase с использованием современных методов шифрования.

5. Защита данных

Мы применяем следующие меры безопасности:

  • Шифрование всех локальных данных с использованием AES-256
  • Защищенное соединение при синхронизации данных (HTTPS/TLS)
  • Аутентификация через Firebase Authentication
  • Регулярные обновления безопасности
  • Двухфакторная аутентификация (при включении)

6. Передача данных третьим лицам

Мы не продаем и не передаем ваши личные данные третьим лицам для маркетинговых целей.

Данные могут быть переданы только:

  • С вашего явного согласия
  • По требованию правоохранительных органов при наличии законного запроса
  • Сервисным партнерам (Firebase, Telegram) для обеспечения работы приложения

7. Файлы cookie и отслеживание

Наше приложение не использует файлы cookie для отслеживания вашей активности. Мы собираем минимально необходимую техническую информацию для обеспечения работы приложения.

8. Права пользователей

Вы имеете право:

  • Получить доступ к вашим личным данным
  • Исправить неточные данные
  • Удалить вашу учетную запись и все связанные данные
  • Экспортировать ваши данные в стандартном формате
  • Отозвать согласие на обработку данных

Для реализации ваших прав свяжитесь с нами по адресу: support@shoga.app

9. Данные несовершеннолетних

Наше приложение не предназначено для лиц младше 16 лет. Мы сознательно не собираем личную информацию от несовершеннолетних. Если вы обнаружили, что несовершеннолетний предоставил нам личные данные, свяжитесь с нами для удаления этой информации.

10. Международная передача данных

Ваши данные могут обрабатываться на серверах, расположенных в разных странах. Мы обеспечиваем адекватный уровень защиты данных при международной передаче в соответствии с применимым законодательством.

11. Изменения в Политике

Мы можем периодически обновлять настоящую Политику конфиденциальности. О существенных изменениях мы уведомим вас через приложение или по электронной почте. Дата последнего обновления указана в начале документа.
