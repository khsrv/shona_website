FAQ

Частые вопросы

Для кого подходит Hisob?

Для владельцев торговых точек, небольших магазинов и предпринимателей.

Нужен ли интернет для работы?

Нет. Основные функции работают офлайн, а при подключении можно сделать синхронизацию.

Безопасны ли данные?

Да. Данные шифруются, хранятся локально, резервные копии доступны только владельцу.

Есть ли функции кассы и сканирования?

Да. Есть продажи, сканер штрих-кодов и работа с товарами.

Сколько стоит использование?

Для новых клиентов есть 14 дней пробного периода. Дальше можно выбрать подписку на удобный срок, без больших стартовых платежей.
